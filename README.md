# Embedded Calculator Keil Project

This repository contains an embedded calculator project developed in Keil µVision.

The project uses a keypad for user input, an LCD for displaying values and results, a PLL module for clock configuration, and calculation functions for arithmetic operations.

## Project Files

- `main.c`  
  Main program file. It controls the calculator flow and connects the keypad, LCD, PLL, and calculation modules.

- `Calculations.c` and `Calculations.h`  
  Contains the calculator logic and arithmetic operations.

- `KeyPad.c` and `KeyPad.h`  
  Contains the keypad setup and key-reading functions.

- `LCD.c` and `LCD.h`  
  Contains the LCD setup and display functions.

- `PLL.c` and `PLL.h`  
  Contains the clock setup for the microcontroller.

- `ELEC3362_Active_ch.uvprojx`  
  Keil µVision project file. Open this file in Keil to run/build the project.
